Linux 修改密码

passwd 命令语法

$ passwd --help
Usage: passwd [OPTION...] <accountName>
  -k, --keep-tokens       keep non-expired authentication tokens
  -d, --delete            delete the password for the named account (root only)
  -l, --lock              lock the password for the named account (root only)
  -u, --unlock            unlock the password for the named account (root only)
  -e, --expire            expire the password for the named account (root only)
  -f, --force             force operation
  -x, --maximum=DAYS      maximum password lifetime (root only)
  -n, --minimum=DAYS      minimum password lifetime (root only)
  -w, --warning=DAYS      number of days warning users receives before password
                          expiration (root only)
  -i, --inactive=DAYS     number of days after password expiration when an account
                          becomes disabled (root only)
  -S, --status            report password status on the named account (root only)
  --stdin                 read new tokens from stdin (root only)

Help options:
  -?, --help              Show this help message
  --usage                 Display brief usage message

修改用户密码

修改当前用户(root)的密码：

$ passwd
Changing password for user root.
New password: 
BAD PASSWORD: The password is a palindrome
Retype new password: 
passwd: all authentication tokens updated successfully.

修改其他用户(tom)的密码：

$ passwd tom
Changing password for user tom.
New password: 
BAD PASSWORD: The password is a palindrome
Retype new password: 
passwd: all authentication tokens updated successfully.

查看其他用户(tom)的密码：

$ passwd -S tom
tom PS 1970-01-01 0 99999 7 -1 (Password set, SHA512 crypt.)

锁定其他用户(tom)的密码：

$ passwd -l tom
Locking password for user tom.
passwd: Success

解锁其他用户(tom)的密码：

$ passwd -u tom
Unlocking password for user tom.
passwd: Success

设置用户(tom)最长密码有效期：

$ passwd -x 90 tom
Adjusting aging data for user tom.
passwd: Success
# 设置 tom 用户的最长有效期为 90 天

设置用户(tom)最短密码有效期：

$ passwd -n 30 tom
Adjusting aging data for user tom.
passwd: Success
# 设置 tom 用户的最短有效期为 30 天

密码到期前用户(tom)收到警告的天数：

$ passwd -w 7 tom
Adjusting aging data for user tom.
passwd: Success
# 设置 tom 用户密码到期前7天(默认)收到警告

设置用户(tom)的密码过期：

$ passwd -e tom
Expiring password for user tom.
passwd: Success

密码过期后账户被禁用的天数：

$ passwd -i -1 tom
Adjusting aging data for user tom.
passwd: Success

删除用户(tom)的密码：

$ passwd -d tom
Removing password for user tom.
passwd: Success
$ chage -l tom
Last password change		: Feb 26, 2022
Password expires									: May 27, 2022
Password inactive									: never
Account expires										: never
Minimum number of days between password change		: 30
Maximum number of days between password change		: 90
Number of days of warning before password expires	: 7

passwd 中文语法：

$ passwd --help
Usage: passwd [选项...] <帐户名称>
  -k, --keep-tokens       保留未过期的身份验证令牌
  -d, --delete            删除指定帐户的密码（仅限 root）
  -l, --lock              锁定指定帐户的密码（仅限 root）
  -u, --unlock            解锁指定帐户的密码（仅限 root）
  -e, --expire            使指定帐户的密码过期（仅限 root）
  -f, --force             强制操作
  -x, --maximum=DAYS      最长密码有效期（仅限 root）
  -n, --minimum=DAYS      最短密码有效期（仅限 root）
  -w, --warning=DAYS      密码到期前用户收到警告的天数（仅限 root）
  -i, --inactive=DAYS     密码过期后帐户被禁用的天数（仅限 root）
  -S, --status            报告指定帐户的密码状态（仅限 root）
  --stdin                 从标准输入读取新令牌（仅限 root）

Help options:
  -?, --help              显示此帮助消息
  --usage                 显示简要使用信息

单用户修改 root 密码

重启机器，当开机出现引导菜单时，按 e 进入编辑界面。

init=/sysroot/bin/sh
Ctrl + X
chroot /sysroot/
mount -o remount,rw /
passwd
touch / .autorelabel
Ctrl + D
reboot
