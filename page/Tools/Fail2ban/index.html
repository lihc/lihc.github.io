Fail2ban 工具

项目地址：https://github.com/fail2ban/fail2ban

Fail2ban 的安装

# 使用 yum 方式安装
yum -y install epel-release
yum -y install fail2ban

Fail2ban 的配置

Fail2Ban 配置文件目录

/etc/fail2ban/
├── action.d    # 动作目录
├── fail2ban.conf    # fail2ban 主配置文件
├── fail2ban.d    # fail2ban 配置目录
├── filter.d
├── jail.conf    # 监禁主配置文件
├── jail.d    # 监禁配置目录
├── paths-common.conf
└── paths-fedora.conf

复制 Fail2ban 配置文件

cp /etc/fail2ban/fail2ban.conf /etc/fail2ban/fail2ban.local
cp /etc/fail2ban/jail.conf /etc/fail2ban/jail.local
# 使用修改 *.local 配置文件，将覆盖 *.conf 中的配置。

启动 Fail2ban 服务

systemctl enable --now fail2ban.service

查看 Fail2ban-client 配置

$ fail2ban-client -d
['set', 'syslogsocket', 'auto']
['set', 'loglevel', 'INFO']
['set', 'logtarget', '/var/log/fail2ban.log']
['set', 'dbfile', '/var/lib/fail2ban/fail2ban.sqlite3']
['set', 'dbmaxmatches', 10]
['set', 'dbpurgeage', '1d']

查看 Fail2ban-client 状态

$ fail2ban-client status

`- Jail list:

Fail2ban 的使用

配置 sshd 拦截规则

当使用 ssh 服务远程本机 10 分钟内连续输入密码失败 3 次，自动封堵 IP 一天。

配置 jail 规则

cat <<EOF > /etc/fail2ban/jail.d/sshd.conf
[sshd]
enabled = true
bantime = 1d
filter = sshd
findtime = 10m
action = iptables[name=SSH, port=ssh, protocol=tcp]
logpath = /var/log/secure
maxretry = 3
EOF
  • bantime：禁止时间
  • findtime：持续时间
  • action：应用动作
  • logpath：日志文件
  • maxretry：最大失败次数

重新加载配置

fail2ban-client reload

fail2ban-client set sshd banip 202.79.173.232
