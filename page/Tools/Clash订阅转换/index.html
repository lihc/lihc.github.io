搭建 Clash 订阅转换
白羊
 2021-07-09 06:30:18  
  • Tools 
 
  • Clash 
  • | Sub-web 
  • | Subconverter 
 

Github 项目地址：

  • Sub-web：https://github.com/CareyWang/sub-web

  • Subconverter：https://github.com/tindy2013/subconverter

前端 Sub-web

安装 Nginx 服务

export NGINX_VERSION=1.20.1
curl https://hellolisihui.com/code/nginx_install.sh | bash

下载 Sub-web 文件

yum -y install git
cd /usr/local/nginx/html
git clone https://github.com/CareyWang/sub-web.git

安装 Node.js

cd /root
wget https://nodejs.org/dist/latest-v12.x/node-v12.22.3-linux-x64.tar.xz
tar Jxf node-v12.22.3-linux-x64.tar.xz -C /usr/local/nodejs

echo 'export PATH=/usr/local/nodejs/bin:$PATH' >> /etc/profile
export PATH=/usr/local/nodejs/bin:$PATH

配置前端文件

vim /usr/local/nginx/html/sub-web/src/views/Subconverter.vue

修改 258 行，backendOptions 参数

# 修改为自己的后端地址
backendOptions: [{ value: "https://sub.hellolisihui.com/sub?" }],

生成前端文件

cd /usr/local/nginx/html/sub-web
npm -g install yarn
yarn install
yarn build

后端 Subconverter

下载 Subconverter 文件

wget https://github.com/tindy2013/subconverter/releases/download/v0.6.4/subconverter_linux64.tar.gz

解压缩到目标目录

tar zxf subconverter_linux64.tar.gz  -C /usr/local

添加 Subconverter 服务

cat <<EOF > /etc/systemd/system/subconverter.service
[Unit]
Description=A API For Subscription Convert
After=network.target
 
[Service]
Type=simple
ExecStart=/usr/local/subconverter/subconverter
WorkingDirectory=/usr/local/subconverter
Restart=always
RestartSec=10
 
[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl start subconverter

修改 Nginx 配置文件

server {
    listen       80;
    server_name  sub.hellolisihui.com;

    access_log  logs/sub.access.log;

    root html/sub-web/dist;
    index index.html index.htm;

    error_page 404 /index.html;

    gzip on; #开启gzip压缩
    gzip_min_length 1k; #设置对数据启用压缩的最少字节数
    gzip_buffers 4 16k;
    gzip_http_version 1.0;
    gzip_comp_level 6; #设置数据的压缩等级,等级为1-9，压缩比从小到大
    gzip_types text/plain text/css text/javascript application/json application/javascript application/x-javascript application/xml; #设置需要压缩的数据格式
    gzip_vary on;

    location ~* \.(css|js|png|jpg|jpeg|gif|gz|svg|mp4|ogg|ogv|webm|htc|xml|woff)$ {
        access_log  off;
        add_header  Cache-Control "public,max-age=30*24*3600";
    }

    location /sub {
        access_log  off;
        proxy_pass  http://127.0.0.1:25500/sub;
    }
}

systemctl start nginx

访问 Nginx 服务即可。
