Hexo+Github 搭建博客

一、环境部署

安装 Git 命令

Git 官方网站：https://git-scm.com/

1、在 Windows 上安装：

下载 Git 安装包双击运行，按照提示安装即可。

下载地址：https://github.com/git-for-windows/git/releases


2、在 Linux 上安装：

官网文档：https://git-scm.com/download/linux

# CentOS
yum -y install git

安装 Node.js

Node.js 官方网站：https://nodejs.org/zh-cn/

1、在 Windows 上安装：

下载 Node.js 安装包双击运行，安装提示安装即可。

下载地址：https://nodejs.org/zh-cn/download/releases/

推荐 Node.js 12.x 版本。软件名格式：node-v12.22.2-x64.msi


2、在 Linux 上安装：

在 Linux 上使用二进制安装：https://github.com/nodejs/help/wiki/Installation

#!/usr/bin/env bash
VERSION=${VERSION:-'v12.22.2'}
DISTRO=${DISTRO:-linux-x64}
wget https://nodejs.org/download/release/"$VERSION"/node-"$VERSION"-"$DISTRO".tar.xz
mkdir -p /usr/local/lib/nodejs
tar -xf node-"$VERSION"-"$DISTRO".tar.xz -C /usr/local/lib/nodejs --strip-components=1
# echo 'export PATH=/usr/local/lib/nodejs/bin:$PATH' >> /etc/profile
#source /etc/profile

执行脚本后，运行命令：

echo 'export PATH=/usr/local/lib/nodejs/bin:$PATH' >> /etc/profile
source /etc/profile

二、安装 Hexo

Hexo 官方网站：https://hexo.io/zh-cn/


初始化 Hexo

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

如果多次因为网络原因安装失败，可执行 npm install -g cnpm --registry=https://registry.npm.taobao.org命令。将 npm 命令改为 cnpm 命令执行。

Hexo 目录结构如下：

_config.landscape.yml  db.json       package.json       scaffolds  themes
_config.yml            node_modules  package-lock.json  source

执行命令

hexo clean && hexo g

目录结构如下：

_config.landscape.yml  db.json       package.json       public     source
_config.yml            node_modules  package-lock.json  scaffolds  themes
  • _config.landscape.yml：主题配置文件。landscape 为主题名。
  • _config.yml：Hexo 配置文件。
  • node_modules：依赖包文件夹。
  • public：公共文件夹，用于存放生成的站点文件。
  • scaffolds：模板文件夹。
  • source：资源文件夹，用来存放内容。
  • themes：主题文件夹，用来存放主题文件。

绑定 Github

1、点击链接，创建 Github 账号。已有账号 登录。

2、创建仓库。创建一个仓库名 (Repository name) 为：账号名+github.io 的公有仓库。点击 Create repository 按钮进行创建。点击 SSH 按钮，获得仓库地址。

3、修改配置文件_config.yml。

# _config.yml
...
# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
- type: 'git'
  repo: 'git@github.com:账户名/账户名.github.io.git'
  branch: master

点击 SSH 按钮，即可获得仓库地址。

4、创建 SSH 公钥。输入命令回车：

ssh-keygen

查看 SSH 公钥：

5、安装部署命令：

npm install hexo-deployer-git --save

6、Git 命令绑定 Github 账号：

git config --global user.name "USER_NAME"
git config --global user.email "USER_EMAIL"

7、执行上传命令：

hexo clean # 清除缓存(删除public文件夹)
hexo g # 生成页面(生成页面存放在public下)
hexo d # 上传命令(将public下的文件上传到Github仓库)

8、访问站点 https://.github.io

Demo：https://hellolisihui.github.io

三、更换主题

Hexo 主题地址：https://hexo.io/themes/

挑选合适的主题，参照主题使用文档进行更换。

下载主题

使用博客主题 Keep 为例。

如果正在使用 Hexo 5.0 或更高版本，使用 npm 安装：

cd blog
npm install hexo-theme-keep

或者使用 Git 克隆整个仓库：

cd blog
git clone https://github.com/XPoet/hexo-theme-keep themes/keep

安装完成后修改配置文件：

# _config.yml
theme: keep

主题插件

视频播放插件

npm install hexo-tag-dplayer --save

本地搜索插件

npm install hexo-generator-searchdb -save

RSS 订阅插件

npm install hexo-generator-feed -save

字数统计插件

npm install hexo-wordcount -save

站点地图插件

npm install hexo-generator-sitemap --save

文章加密插件

npm install --save hexo-blog-encrypt

资源压缩插件

npm install hexo-all-minifier --save
npm install mozjpeg --save
npm install optipng-bin --save
npm install jpegtran-bin --save

数学公式插件

npm install hexo-filter-mathjax --save

部署命令

npm install hexo-deployer-git --save
