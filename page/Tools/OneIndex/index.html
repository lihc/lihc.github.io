搭建 OneIndex 个人网盘

OneIndex：OneDrive 目录索引。

安装 Nginx 服务

使用脚本进行安装

export NGINX_VERSION=1.18.0
curl https://hellolisihui.com/code/nginx_install.sh | bash

安装 PHP 环境

解决依赖环境

yum -y install gcc gcc-c++ libxml2 libxml2-devel curl curl-devel wget sqlite-devel

下载 PHP 源码

wget https://www.php.net/distributions/php-7.4.21.tar.gz

解压缩 PHP

tar zxf php-7.4.21.tar.gz -C /usr/local/src/

源码编译 PHP

# 开启php-fpm模块, 添加curl支持
cd /usr/local/src/php-7.4.21/
./configure --enable-fpm --with-curl
# 若内存不足 1024MB, 添加 "--disable-fileinfo" 参数

编译安装 PHP

make
make install

调整 PHP 配置文件

cp php.ini-development /usr/local/php/php.ini
cp /usr/local/etc/php-fpm.d/www.conf.default /usr/local/etc/php-fpm.d/www.conf
cp sapi/fpm/php-fpm /usr/local/bin

如果文件不存在，则阻止 Nginx 将请求发送到后端的 PHP-FPM 模块

vim /usr/local/php/php.ini
cgi.fix_pathinfo=0

修改 PHP 配置文件

cp /usr/local/etc/php-fpm.conf.default /usr/local/etc/php-fpm.conf
vim /usr/local/etc/php-fpm.conf
#最后一行
include=/usr/local/etc/php-fpm.d/*.conf

启动 php-fpm 服务

/usr/local/bin/php-fpm

下载所需文件

下载 OneIndex 文件

cd /usr/local/nginx/html
yum -y install git
git clone https://github.com/hellolisihui/oneindex.git

修改目录权限

chmod 777 /usr/local/nginx/html/oneindex/config
chmod 777 /usr/local/nginx/html/oneindex/cache

修改 Nginx 配置

vim /usr/local/nginx/conf/nginx.conf
server {
    listen       80;
    server_name  localhost;

    root   html/oneindex;
    index  index.php index.html index.htm;

    access_log  logs/index.access.log;

    location ~ \.php$ {
        root           html/oneindex;
        fastcgi_pass   127.0.0.1:9000;
        fastcgi_index  index.php;
        fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;
        include        fastcgi_params;
    }

    # Nginx 伪静态
    if (!-f $request_filename){
        set $rule_0 1$rule_0;
    }
    if (!-d $request_filename){
        set $rule_0 2$rule_0;
    }
    if ($rule_0 = "21"){
        rewrite ^/(.*) /?/$1 last;
    }

}

启动 Nginx 服务

systemctl start nginx

安装 OneIndex
