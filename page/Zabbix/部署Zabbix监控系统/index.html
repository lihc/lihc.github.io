部署Zabbix监控系统
白羊
 2020-07-03 12:02:41  
  • Zabbix 
 
  • Zabbix 
 

Zabbix中文官网：https://www.zabbix.com/cn/

Zabbix下载&文档：https://www.zabbix.com/cn/download

1. 选择Zabbix服务器平台

Zabbix版本OS数据库Web Server
4.0 LTSCentOS 7MySQLApache

2. 配置Yum源

rpm -Uvh https://repo.zabbix.com/zabbix/4.0/rhel/7/x86_64/zabbix-release-4.0-2.el7.noarch.rpm

3. 安装服务器端软件

yum -y install zabbix-server-mysql zabbix-web-mysql zabbix-agent  mariadb-server

4. 创建初始数据库

systemctl start mariadb && systemctl enable mariadb
MariaDB [(none)]> 
create database zabbix character set utf8 collate utf8_bin;
create user zabbix@localhost identified by 'zabbix';
grant all privileges on zabbix.* to zabbix@localhost;

数据导入Zabbix数据库

zcat /usr/share/doc/zabbix-server-mysql*/create.sql.gz | mysql -uzabbix -p zabbix
Enter password:  # 输入Zabbix数据库密码(zabbix)

5. Zabbix修改配置文件

配置数据库

vim /etc/zabbix/zabbix_server.conf
DBPassword=zabbix

配置时区

vim /etc/httpd/conf.d/zabbix.conf
php_value date.timezone Asia/Shanghai

6. 启动Zabbix进程

启动Zabbix Server和Agent进程

systemctl restart zabbix-server zabbix-agent httpd
systemctl enable zabbix-server zabbix-agent httpd

通过Web界面初始化配置Zabbix：http://Server-IP-or-Name/zabbix

根据提示，填写数据库密码，然后，一律下一步。

配置完成登录：用户名：Admin 密 码：zabbix

修改中文显示：Administration – Users – Admin – Language – Chinese – Update

7. 客户端配置监控

开启CentOS客户端，测试Zabbix监控

安装Zabbix-Agent

rpm -ivh https://repo.zabbix.com/zabbix/4.0/rhel/7/x86_64/zabbix-agent-4.0.0-2.el7.x86_64.rpm
rpm -ivh https://mirrors.aliyun.com/zabbix/zabbix/4.0/rhel/7/x86_64/zabbix-agent-4.0.0-2.el7.x86_64.rpm

修改Agent配置文件，指向Zabbix-Server的ip地址

vim /etc/zabbix/zabbix_agentd.conf
Server=服务端IP
systemctl start zabbix-agent
systemctl enable zabbix-agent

添加主机：配置–主机–创建主机

在Zabbix-Server上修改图形中文乱码情况

yum -y install wqy-microhei-fonts
cp /usr/share/fonts/wqy-microhei/wqy-microhei.ttc /usr/share/zabbix/assets/fonts/graphfont.ttf
